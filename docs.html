How to use vebscraper

  1. Step 1: Copy the URL of Your Desired Web Page and Paste It in the Input Box.
  1. Step 2: Click on the Scrape Button
  1. Now, you will see the Web Scraping result of the page.

What results this scraping tool provides

This instant data scraper offers the following results:

  1. Title: Webpage title and title length (in characters)
  2. Meta Description: Meta description and description length (in characters)
  3. Headings: Count and list of h1-h6 headings
  4. Images: Total image count with image paths and alt text
  5. Internal Links: Total internal links count and list
  6. External Links: Total external links count and list
  7. Canonical Tag: Checks if the page has a canonical tag and provides the canonical link if present
  8. Open Graph Tags: OG Title, OG Description, OG Type, OG URL
  9. Page Data: Page size, total words on the page, total paragraphs
  10. Social Media Links: Social media links such as Facebook, LinkedIn, Twitter
  11. HTTPS: Checks if the site uses HTTPS or not
  12. Common Words: One word and Two Word Phrases that are repeated more than 3 times (Maximum 20)
  13. Security Measurements: Checks 10 Security Measurements
